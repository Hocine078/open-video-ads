Open Video Ads VAST Actionscript 3 Framework - Version 0.3.3

Welcome to the (Beta) Open Video Ads VAST Actionscript 3 Framework.

This package contains the source (AS3) files required create a custom player implementation that is VAST compliant.

Where required, use "Ant" to build this release and create a new SWC library.

The required build files are included with the package - to build, use "ant build" from the package root directory.

Further documentation can be found on the detailed documentation pages at www.openvideoads.org or the google code repository supporting the framework - code.google.com/p/open-video-ads
