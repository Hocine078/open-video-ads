Open Video Ad Streamer for Flowplayer - Beta Version 0.2.4

Welcome to the beta version (0.2.4) of the Open Video Ad Streamer for Flowplayer.

This package contains the binary (SWF) files required to deploy the plugin with Flowplayer 3.1.3 +.

The package also contains the source files and ant build files.

Several examples are also provided to help illustrate the various configurations for the plugin.

Further documentation can be found on the detailed plugin documentation pages at the google code repository code.google.com/p/open-video-ads
