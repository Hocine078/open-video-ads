Open Video Ad Streamer for JW Player - Beta Version 0.3.3

Welcome to the beta version (0.3.3) of the Open Video Ad Streamer for Longtail Video's JW Player.

This package contains the binary (SWF) files required to deploy the plugin with JW Player 4.5 from Longtail Video.

Several examples are also provided to help illustrate the various configurations for the plugin.

Further documentation can be found on the detailed plugin documentation pages at the google code repository open-video-ads.googlecode.com
