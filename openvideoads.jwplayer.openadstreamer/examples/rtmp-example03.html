Example 3 (RTMP) - A pre-roll and show stream with a (video) overlay ad

A 30 second pre-roll ad plays followed by the show stream. 5 seconds into the show stream an overlay appears for 15 seconds - click on the overlay to start up a video ad. Once the video ad has played, the show stream resumes at the point at which the overlay was clicked.


The configuration for this example is: 


<embed 
  id="ply"
  data="../dist/4.5.swf"
  src="../dist/4.5.swf" 
  width="450"
  height="300"
  allowscriptaccess="always"
  allowfullscreen="true"
  flashvars="height=300&width=450&plugins=../dist/OpenAdStreamer.swf&config=config/rtmp03.xml"
/>

"rtmp02.xml" is defined as: 


<config>
   <openadstreamer.title>
       RTMP Example 03 - Pre-Roll and Overlay
   </openadstreamer.title>
   <openadstreamer.json>
       {
            "allowPlaylistControl": false,

            "shows": {
               "baseURL": "rtmp://ne7c0nwbit.rtmphost.com/videoplayer",
               "streams": [
                   {
                       "file":"the-black-hole.mp4",
                       "duration":"00:00:35"
                   }
               ]
            },

            "overlays": {
               "regions": [
                   {
                       "id": "bottom",
                       "verticalAlign": "bottom",
                       "horizontalAlign": "center",

            },
